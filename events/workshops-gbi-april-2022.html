Organisers: Global BioImaging

Location: held remotely

Workshop: OME-NGFF: Towards a community standard image file format for sharing big image data in the cloud

Date: 6 April 2022

Duration: 3 hours

Recent work on a next-generation format that aims to enable a one-time conversion of datasets to a common format was presented. This workshop served as an introduction to OME-NGFF file format and hence, was suitable for beginners to this topic. For more details please visit OME-NGFF workshop.
