Organisers: Canada BioImaging

Location: McGill University, Montreal, Canada

Workshop: OMERO Hands-on Workshop

Dates: 31st July - 1st August 2023

OMERO, a component of OME, is a software platform for image data management and analysis which will be presented in this workshop. Import, organisation, viewing, searching, annotation and publishing of imaging data with OMERO will be shown.

Further, the emphasis will be given on the popular OMERO.figure application.

During the second day, we will be concentrating on more advanced OMERO topics, such as using OMERO Python API for running image analysis on the cloud. In the afternoon, a discussion about automatic metadata import strategies into OMERO will be held.
