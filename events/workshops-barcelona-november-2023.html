Organisers: Centre for Genomic Regulation, Barcelona.

Location: Centre for Genomic Regulation (CRG), Barcelona.

Workshop: OMERO workshop.

Dates: 7 - 8 November 2023

OMERO, a component of OME, is a software platform for image data management and analysis which will be presented in this workshop. Import, organisation, viewing, searching, annotation and publishing of imaging data with OMERO will be shown.

Further, the emphasis was given on the popular OMERO.figure application.

During the second day, we will be concentrating on more advanced OMERO topics, showing connections of OMERO with popular 3rd party image analysis software packages such as Fiji, Cellprofiler, R, Python and using OMERO Python API for running image analysis on the cloud.
