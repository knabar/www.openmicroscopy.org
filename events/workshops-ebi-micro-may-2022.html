Organisers: Microscopy data analysis course

Location: held remotely

Workshop on Day 3: Distributed/Cloud computing - practical

Date: 18 May 2022

Workshop on Day 4: Image data resource - practical

Date: 19 May 2022

Duration: 2 hours (Distributed computing workshop) and 1 hour (IDR workshop)

The Distributed/Cloud computing workshop ntroduced an emerging cloud-compatible imaging format currently used by the BioImage Archive and the Image Data Resource. It was shown how it can be used to access cloud-hosted imaging data and how it facilitates analysis in parallel. In the second workshop on the following day, the Image Data Resource (IDR) was explored and it was demonstrated how to access the IDR data using the Python API, to fetch image data and metadata from IDR study associated with a publication. Further, re-analysis of data using an alternative segmentation tool to the one used by the publication authors was performed.
